# Project 2 Sprint 3 Planning

Deze sprint is van maandag 4 november tot vrijdag 17 november.

In deze planning staat de user story planning van sprint 3.

## Planning van taken

Wij hebben een inschatting gemaakt van hoeveel tijd we nodig gaan hebben voor de taken van de sprint. Hierin hebben wij een prioritering gemaakt van de user stories zodat we goed weten waar we opvolgorde aan moeten werken. Als we binnen een bepaalde tijd die we hebben gegeven per user story er langer of korter over doen, dan passen we het aan in de planning, zodat we de planning kunnen verschuiven en we de haalbaarheid van de sprint beter kunnen inschatten.
